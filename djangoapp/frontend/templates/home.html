Features

Viewer

The DYSSE Viewer allows for the visualization of the 3D model along with a dynamic dimension table, as well as offering configuration options and detailed document information.

IMS Viewer

The IMS Viewer displays the values filled in via IMS, integrating them with the DYSSE 3D model for comprehensive analysis.

Control Dimension Selection

A tool for selecting the control dimensions that will be used in IMS test, ensuring precision and efficiency in the manufacturing process.

Connection 3D

The Connection3D is designed to assist operators in making the connections of a wound stator.

List of Characteristics

A window for registering characteristics and components
